ZEROMEANER

WHAT

Zeromeaner is a fork of the popular Tetris clone Nullpomino. However, Zeromeaner will run in a web browser. It is also now a maven project hosted on a git repository.

WHY

Nullpomino hasn't seen a release for almost two years now, and I got impatient. Zeromeaner will follow the open-source mantra of "release early, release often". When new features are developed, you will actually get them!

HOW

I used git to make a clone of the Nullpomino subversion repository. I then converted the project to use maven. (That conversion is mostly complete; I'm not sure about the native library support yet.) Then I added Java applet support.

WHEN

The fork began on Nov 25, 2012.

WHERE

You can download a standalone version of Zeromeaner from here. You can download the source code from github.
